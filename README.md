# MelodyMelonTap v2.26.1 Universal Field

เวอร์ชันนี้ต่อยอดจาก v2.26.0 CleanHybrid เพื่อให้เหมาะกับการใช้งานจริงหลายสภาพแวดล้อม เช่น สวน ตลาด ห้าง ร้านผลไม้ และข้างทาง

## จุดสำคัญ
- ใช้ช่วงวิเคราะห์เสียงหลัก 100–1000 Hz
- ใช้ V1 Ripeness Engine สำหรับระดับการสุก
- เคาะ 3 ครั้ง แล้วใช้ค่าเฉลี่ย f_peak เป็นตัวตัดสินระดับการสุก
- แสดงระดับการสุกเป็น ดิบ / ใกล้สุก / สุกพอดี / อาจเน่า/สุกเกิน
- ไม่แสดงกลุ่ม A–E บนหน้าแอป
- เพิ่ม Auto Noise Calibration ก่อนเริ่มวัด
- เพิ่ม Background Spectrum Subtraction เพื่อลดเสียงรบกวนรอบข้าง
- เพิ่มความไวเสียงแบบ อัตโนมัติ / ต่ำ / ปกติ / สูง
- เพิ่ม progress การเคาะ 1/3, 2/3, 3/3
- เพิ่มเกจตำแหน่งความถี่ระดับการสุก
- เพิ่มปุ่มวัดซ้ำลูกนี้
- หยุดไมโครโฟนหลังวิเคราะห์เสร็จ
- ความหวาน ความกรอบ ความฉ่ำ และความเสี่ยงโพรง ยังใช้โมเดลหลายฟีเจอร์เสียงจาก v2.25.1 เป็นฐาน
- ความกรอบจำกัดค่าขั้นต่ำที่ 5 N
- ถ้าเสี่ยงโพรงชัดเจน จะจัดอยู่ในกลุ่ม อาจเน่า/สุกเกิน
- AI Voice จะอ่านเรื่องโพรงเฉพาะเมื่อมีความเสี่ยงโพรงเท่านั้น
- มีประวัติผลและ Export CSV

## หมายเหตุ
ผลวิเคราะห์เป็นการประเมินจากเสียงเคาะ ควรใช้ร่วมกับการสังเกตภายนอกและการเคาะซ้ำเมื่อผลอยู่ใกล้เส้นแบ่งหรือสัญญาณเสียงไม่ชัด
